# Assets

Drop the looping training background video here as:

```
ballista-training-bg.mp4
```

The landing page references `/assets/ballista-training-bg.mp4`. If the file is
absent, the page still renders correctly — the dark gradient overlays act as the
fallback background. Recommended: a short, muted, dark/cinematic baseball-training
clip, H.264 MP4, ~1080p, kept small for fast load.
